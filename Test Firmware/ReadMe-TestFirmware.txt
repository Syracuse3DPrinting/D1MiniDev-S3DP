Test Firmware is ESPHome with no internet connectivity. 

This tests full functionality of the module without needing to be connected to your Home Assistant instance. 

To use this firmware:

Download ESPHome Flasher https://github.com/esphome/esphome-flasher/releases

Plug in ESP8266 via USB. Select COM port. Select firmware QC file. 

Once install is complete log window should display button press changes and temperature readings. 

LED should cycle colors in a test pattern.
